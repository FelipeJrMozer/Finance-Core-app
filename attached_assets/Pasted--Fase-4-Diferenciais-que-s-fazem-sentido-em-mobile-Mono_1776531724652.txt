## 1. QR Scanner — Pix (BR Code) e NFe
**Lib a adicionar**: `expo-camera` (versão compatível com Expo SDK 54: `~17.0.x`).
**Novo arquivo**: `artifacts/finance-core/app/scan.tsx`
- Tela full-screen com câmera traseira e área de mira centralizada.
- Permissão explícita; se negada, instrução clara para abrir as configurações.
- Detecta QR e tenta classificar:
  - **Pix BR Code** (EMV padrão BCB): começa com `00020126` ou `00020101`. Decodificar TLV (tag-length-value):
    - tag 26 → payload Pix (chave, valor sugerido)
    - tag 54 → valor (string com 2 decimais)
    - tag 59 → nome do beneficiário
    - tag 60 → cidade
    - tag 62 → infos adicionais (txid)
  - **Chave NFe** (44 dígitos numéricos com ou sem prefixo `NFe`): extrair com regex `[0-9]{44}`.
  - Caso contrário: mostrar valor cru e botão "Copiar".
- Após detectar Pix: roteia para `transaction/add` com query params pré-preenchidos: `description`, `amount`, `notes` (incluir nome+cidade do beneficiário). Tipo = expense, conta = última usada.
- Após detectar NFe: roteia para `transaction/add` com `notes` contendo a chave; deixar valor/descrição em branco para o usuário preencher (não temos endpoint de consulta SEFAZ).
**Helper**: `artifacts/finance-core/utils/pixDecoder.ts` com função `decodePixCode(code: string): { merchantName?, city?, amount?, txid?, key? } | null`.
**Acesso**: novo botão "Escanear" no `QuickTransactionModal` (já existe), ao lado de "Adicionar". Também atalho no Dashboard.

Helper: artifacts/finance-core/utils/smsParser.ts com regex para principais bancos brasileiros:

type ParsedSms = { bank?: string; amount?: number; type?: 'expense'|'income'; description?: string; cardLast4?: string };
const PATTERNS: Array<{ bank: string; regex: RegExp; type: 'expense'|'income'; map: (m: RegExpMatchArray) => Partial<ParsedSms> }> = [
  // Nubank
  { bank: 'Nubank', type: 'expense',
    regex: /Compra aprovada.*?R\$\s?([\d.,]+).*?em\s+([^\.]+?)(?:\.|\s+em)/i,
    map: m => ({ amount: parseBRL(m[1]), description: m[2].trim() }) },
  // Itaú
  { bank: 'Itaú', type: 'expense',
    regex: /compra.*?R\$\s?([\d.,]+).*?cart[aã]o\s+final\s+(\d{4})/i,
    map: m => ({ amount: parseBRL(m[1]), cardLast4: m[2] }) },
  // Pix recebido
  { bank: 'Pix', type: 'income',
    regex: /pix.*?recebido.*?R\$\s?([\d.,]+).*?de\s+([^\.]+)/i,
    map: m => ({ amount: parseBRL(m[1]), description: `Pix de ${m[2].trim()}` }) },
  // Pix enviado
  { bank: 'Pix', type: 'expense',
    regex: /pix.*?(enviado|realizado).*?R\$\s?([\d.,]+).*?para\s+([^\.]+)/i,
    map: m => ({ amount: parseBRL(m[2]), description: `Pix para ${m[3].trim()}` }) },
];
export function parseBRL(s: string): number {
  return parseFloat(s.replace(/\./g, '').replace(',', '.'));
}
export function parseBankSms(text: string): ParsedSms | null {
  for (const p of PATTERNS) {
    const m = text.match(p.regex);
    if (m) return { bank: p.bank, type: p.type, ...p.map(m) };
  }
  return null;
}
