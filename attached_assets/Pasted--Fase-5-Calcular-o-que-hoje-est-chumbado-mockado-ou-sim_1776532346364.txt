## 3. Sortino Ratio (NOVO)
**Mesmo arquivo**.
**Implementação**:
```ts
function sortino(dailyReturns: number[], rfDailyTarget = 0): number | null {
if (dailyReturns.length < 60) return null;
const mean = dailyReturns.reduce((a, b) => a + b, 0) / dailyReturns.length;
const downside = dailyReturns.filter(r => r < rfDailyTarget);
if (downside.length === 0) return null;
const downsideDev = Math.sqrt(
  downside.reduce((sum, r) => sum + Math.pow(r - rfDailyTarget, 2), 0) / dailyReturns.length
);
if (downsideDev === 0) return null;
const annMean = Math.pow(1 + mean, 252) - 1;
const annDownside = downsideDev * Math.sqrt(252);
const rfAnn = getRiskFreeAnnual(); // do marketBenchmarks
return (annMean - rfAnn) / annDownside;
}

6. MWR (Money-Weighted Return / IRR) — NOVO
Mesmo arquivo.

Implementação:

Lista de cashflows com sinal: aporte negativo, resgate/valor final positivo.
Resolver IRR via Newton-Raphson (limite 100 iterações, tolerância 1e-7):
function irr(cashflows: { date: Date; amount: number }[], guess = 0.1): number | null {
  const t0 = cashflows[0].date.getTime();
  const years = (d: Date) => (d.getTime() - t0) / (365 * 86400000);
  const npv = (r: number) => cashflows.reduce((s, cf) => s + cf.amount / Math.pow(1 + r, years(cf.date)), 0);
  const dnpv = (r: number) => cashflows.reduce((s, cf) => s - years(cf.date) * cf.amount / Math.pow(1 + r, years(cf.date) + 1), 0);
  let r = guess;
  for (let i = 0; i < 100; i++) {
    const f = npv(r);
    if (Math.abs(f) < 1e-7) return r;
    const df = dnpv(r);
    if (df === 0) return null;
    r = r - f / df;
    if (r < -0.999) return null;
  }
  return null;
}

Endpoint: GET /api/portfolio/mwr?from=&to= retornando { mwr, cashflows }.
