### CORREÇÃO 1 — Dashboard: Gauge do Health Score exibe escala errada (/1000 em vez de /100)
**Arquivo:** `artifacts/finance-core/app/(tabs)/index.tsx`
Localizar o componente `HealthGauge` (função por volta das primeiras linhas do arquivo).
Dentro dela, encontrar e corrigir dois pontos:
**Ponto 1a — cálculo do pct:**
```typescript
// ANTES:
const pct = Math.min(score / 1000, 1);
// DEPOIS:
const pct = Math.min(score / 100, 1);

Ponto 1b — texto da escala:

// ANTES:
<Text ...>/1000</Text>
// DEPOIS:
<Text ...>/100</Text>

Com /1000, um score de 80 aparece como 8% do gauge (quase vazio), o que é visualmente errado.

  recommendations: (data.components || []).map((c: any) => c.tip).filter(Boolean),

const getBudgetSpent = (budget: Budget) =>
